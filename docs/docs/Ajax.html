Namespace: Ajax

Ajax()

Ajax is a instance of Ajax class generated for R. It allows the XHR JavaScript's Object management.

Methods

load(data) → {Object}

Loads a new XHR object to send and request data from backend. (A server is needed to work with Ajax).
Parameters:
Name Type Description
data Object Object with followin' elements: method, source, restype, data, datatype:[json|xml|txt] and header. onready: callback, onerror: callback, onpreload: callback
Returns:
this Ajax Object for chain pattern.
Type
Object
Example
// This statement will request some files from backend:
let xhr = this.Ajax(), c = console.log

// A plain text:
xhr.load({
	'data': {'source': './test-ajax.txt'}, // Ajax uses by default GET method.
	'onready': (x) => c('Data response (txt):', x), // It is possible customize an onready callback. 
	'onpreload': (x) => c('Loading... ' + (x * 100 / 4) + '%') // A preloader function callback also.
	'onerror': () => c('Oops') // And an error controller function callback too.
});

// For a JSON file:
// restype means frontend waits for a json response type.
xhr.load({
	'data': {'method': 'POST', 'source': './test-ajax.json', 'restype':'json'},
	'onready': (x) => c('Received data: (json)', x)
});

// For a PHP file:
xhr.load({
	'data': {'data':'x', 'method': 'POST', 'source': './test-ajax.php', 'restype':'json'}, // 'data':'x' sends to PHP a string "x".
	'onready': (x) => c('Received data: (php)', x)
});

// The 'data' (JS Object) parameter of 'load' method can have next required and optional values:
{
	'data': { // All information required by Ajax to work.
		'source': './some/backend/file.txt', // Required. The route path to backend file.
		'method': 'POST|GET', // Optional. Default GET.
		'header': 'Any HTTP header', // Optional. Default: 'application/x-www-form-urlencoded'.
		'restype': 'json|text|xml', // Optional. The response value type required. Default text.
		'data': 'any string' // Optional. Any data to send at backend. Example For json: 'x=' + JSON.stringify(jsondata)
		'datatype': 'text|json|xml', // Optional. The data type that is sended to backend.
	},
	'onready': () => {}, // Optional. A custom onready callback.
	'onpreload': () => {}, // Optional. A custom preloader callback.
	'onerror': () => {} // Optional. And a custom error callback.
}
