Class: R

R(propsopt)

Rocfor.js is a Library to perform great frontend interfaces and agile communication with backend.
Rocforjs comes from Reactive Objects and Components with JavaScript. It was entirely written on ES5. When the Library is called on a project, before start it executes its own register function that searches in window scope, if its reference name already is occupied by another library or framework. The main Rocforjs reference is "R" if this name already exists the name "Rocfor" will be disposed to start with the work, if this name already exists "Rocforjs" will be setted as the Library's name, else Rocforjs will not be registered on JavaScript window object. Once "R" or "Rocfor" or "Rocforjs" is registered on window object, The Library is available on the project and now, is possible to begin the work. Through R (or "Rocfor" or "Rocforjs") is possible work with its mehotds Tag, Block, Ajax, SPA or Mdown. To know the current version of Rocforjs, call its property "version".
