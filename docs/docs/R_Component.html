Class: R.Component

R.Component(propsopt)

Rocforjs Component Module, allows the creation of Reactive Objects and Components.
R.Component (or Rocfor.Component or Rocforjs.Component) is a Class that resides in window Object.
Throwgh R.Component you can create any Reactive Component by extending (recomended) any class inheriting all its properties and methods.

setState()

Sets initial value to the Component state.
Example
'use strict'
class MyClass extends R.Component {
	constructor(props) {
		super(props)
		this.mycomponent = this.Component(`I'll be a Reactive Component!`)
		this.state = {'i': 0} // this.state is a property that Component already has as an empty object {}
		this.start_i() // this.state.i now is 1
	}
	start_i () {
		this.setState(prevState => {'i': prevState['i'] + 1})
	}
}

resetState()

Reset to its initial value the Component state
Example
'use strict'
class MyClass extends R.Component {
	constructor(props) {
		super(props)
		this.mycomponent = this.Component(`I'll be a Reactive Component!`)
		this.state = {'i': 0}
		this.start_i() // this.state.i now is 1
		this.reset_i() // this.state.i now is 0
	}
	start_i () {
		this.setState(prevState => {'i': prevState['i'] + 1})
	}
	reset_i () {
		this.resetState() // sets this.state.i at 0;
	}
}
